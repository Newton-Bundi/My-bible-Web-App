About the web page

This web application allows you to read verses from the bible and provide comments and similar bible passages.
The web page initially displays the most well-known verse: John 3:16.
However, the user can then choose a new verse and add comments to it and link it to a similar passage.
The aim of this web page is to have a repository of different interpretations of verses based on the various users.
